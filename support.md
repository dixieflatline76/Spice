# Spice Support

<p class="intro">We are dedicated to making Spice the best wallpaper manager for macOS and Windows. If you encounter any bugs, have feature requests, or need technical assistance, please use our GitHub Issue Tracker.</p>
<p class="intro" style="font-size: 1.05rem; margin-bottom: 2rem;">Before opening a ticket, check the <a href="user_guide.html" style="color: var(--accent-color); font-weight: 600;">User Guide</a> — it covers common questions about setup, providers, keyboard shortcuts, and troubleshooting.</p>

## How to Get Help

The fastest way to get support is to open a new issue on our GitHub repository:

<a href="https://github.com/dixieflatline76/Spice/issues/new/choose" target="_blank" class="support-btn">Open a Support Ticket</a>

When opening an issue, please include:
- Your macOS version (e.g., macOS Sonoma).
- Your hardware model (e.g., M1 MacBook Pro).
- A clear description of the problem or request.
