# Spice Support

We are dedicated to making Spice the best wallpaper manager for macOS. If you encounter any bugs, have feature requests, or need technical assistance, please use our GitHub Issue Tracker.

## How to Get Help

The fastest way to get support is to open a new issue on our GitHub repository:

[**Open a New Support Ticket**](https://github.com/dixieflatline76/Spice/issues/new)

When opening an issue, please include:
- Your macOS version (e.g., macOS Sonoma).
- Your hardware model (e.g., M1 MacBook Pro).
- A clear description of the problem or request.
