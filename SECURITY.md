# Security Policy

## Supported Versions

Only the latest version of Spice is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of Spice seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, please report it privately.
